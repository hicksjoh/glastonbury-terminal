# Paper-trading lock — Codex QA review 2026-04-28

Defense-in-depth: every Alpaca order submission must go through
`assertPaperTrading()` in `src/lib/alpaca.ts`. The guard throws unless
`ALPACA_BASE_URL`'s host is exactly `paper-api.alpaca.markets`, so a
misconfigured Vercel env, copy-paste, or typo cannot route real orders.
